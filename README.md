# PG Stream

PG Stream is a front-end web development learning project. This project is currently a **work in progress** and is not yet finished.

## About The Project

This project is being built to practice HTML and CSS skills by creating a modern user interface for a movie and TV show streaming platform.

### Current Features
- Responsive layout designed with a dark theme aesthetic.
- Top navigation bar with a search icon.
- Categorized media sections including:
  - Featured banner with "Watch Now" and "Watch Later" actions.
  - Latest
  - Recommendations
  - Movies
  - TV Shows
- Sticky bottom navigation footer (Home, Download, Profile).

### Technologies Used
- HTML5
- CSS3 (Vanilla)
- Google Fonts (Roboto Condensed)

### Upcoming Features
- Desktop responsive layout
- Video player page
- User profile page
- Downloads section
- Dedicated Movie and TV Show pages
- And more!

## Status
🚧 **Work In Progress**: This project is solely for learning purposes. Features, interactions, and styles are actively being developed and refined.
